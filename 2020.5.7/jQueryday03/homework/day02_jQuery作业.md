# 每日作业-jQuery 第 02 天

### 1 - 全选案例（加强训练）

- 题目描述

  全选案例，模拟品优购购物车模块中的全选需求：

  1）点击上面的全选，下面全部选中。

  2）下面的 input 如果全部选中，上面的 input 同步选中。

![001](images/001.png)

- 训练目标

  学会使用 jQuery 的属性操作，重点测试 prop() 操作属性。

- 训练提示

  1.分别获取上面和下面的 input

  2.如果上面的选中下面的全选中

  3.如果下面的选中个数为 4 个上面的同步选中

### 2 - 移动选项（加强训练）

- 题目描述

  ![002](images/002.png)
