## this**

谁调用就指向谁

+ 普通函数中 -> window
+ 事件函数中 -> 触发器/调用者
+ 全局作用域 -> window
+ 定时器 -> window
+ 箭头函数 -> 父作用域
+ 对象/数组 -> 对象/数组

### 总结

谁调用就指向谁 -> 谁.函数() 就指向那个谁/ 函数() -> window.函数()

事件中 this指向都是指向调用者

对象中 this指向都是指向对象

箭头函数 没有this指向 它的指向继承自父作用域的指向

其他情况通通都是指向window的
